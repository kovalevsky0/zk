# JavaScript

**JavaScript** is programming language that it described by these criteria's:

* high-level language
* dynamic type system
* multi-paradigm language
* designed by [@BrendanEich](https://twitter.com/BrendanEich)
* conforms to [ECMAScript](ECMAScript.md) specification
