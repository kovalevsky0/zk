# Test Runner

**Test Runner** is a program that runs tests

Required features:

* Run tests in parallel
* Isolation of tests
* Test filtering
* Watch mode (re-running edited tests)
* Coverage
* Reports (html, etc)

### Examples

* In JavaScript
  * [JestJS](JestJS.md)
  * [mocha.js](mocha.js.md)
  * [ava.js](ava.js.md)
